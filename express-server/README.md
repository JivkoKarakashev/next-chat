This is a [Express.js Server](https://expressjs.com/) project initialized with [`npm install express --save`](https://expressjs.com/en/starter/installing.html).

## Getting Started

Run the app with the following command:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
